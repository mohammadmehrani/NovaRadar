# Nova Radar 🛰️🌌

[![Android CI](https://github.com/mohammadmehrani/NovaRadar/actions/workflows/build.yml/badge.svg)](https://github.com/mohammadmehrani/NovaRadar/actions/workflows/build.yml)
[![Latest Release](https://img.shields.io/github/v/release/mohammadmehrani/NovaRadar)](https://github.com/mohammadmehrani/NovaRadar/releases)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Nova Radar** is a highly polished, futuristic network analyzer and security auditing applet built entirely in Kotlin and modern Jetpack Compose. 

Featuring an advanced glassmorphic dashboard design, interactive radar scan sweep visualizers, a local transaction tracking Room database, and dual-language (English / Persian) capability, the applet operates on top of Material Design 3 and implements **dynamic Gemini gradient themes** for pristine visual style.

## 🚀 Getting Started

### Prerequisites
- **Android Studio Jellyfish | Ladybug** (or newer).
- **JDK 17** or newer.

### Development Setup
The Android project is located in the `android/` directory.

1. Clone the repository:
   ```bash
   git clone https://github.com/mohammadmehrani/NovaRadar.git
   ```
2. Open the `android/` folder in Android Studio.
3. Allow Gradle to sync and download dependencies.

## 🛠️ Tech Stack & Architecture

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose (Material 3)
- **Architecture**: MVVM + Clean Architecture
- **Local Storage**: Room Persistence
- **CI/CD**: GitHub Actions
